    # RNDChess Chess Application (Client, Database, UCI Engine) RNDChessClient -------------- Gui interface to RNDChessDB and RNDChessUCI Written primarilty in C++ utilizing QT 5.4 focusing on QT Quick 2.x RNDChessDB ---------- Initially just a plugin for single game and multigame PGN format files. Later adding a plugin to allow working with SCID format chess databases. Other Database support will be added. RNDChessUCI ----------- Two initial Engines planned one written in C/C++ using a simple evaluation and based primarily on the RNDChessClient's internal support move generation. A second UCI engine written using Google's GO Language Others might be added RNDChessBook ------------ Source in Tex format of a companion book as the above applications where written
